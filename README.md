## Installation

```bash
$ npm install
```

## Database
> Install PostgreSQL https://www.postgresql.org/download/

> Create database: **rekrutacja**
```dotenv
POSTGRES_USER=postgres
POSTGRES_DB=rekrutacja
POSTGRES_PASSWORD=root
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
